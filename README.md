# pdf_extract

This repository is to extract tabular data from PDF using the camelot library.

To install camelot:
1) Install Ghostscript. Link: 
https://www.ghostscript.com/download/gsdnld.html

2) pip install camelot
```
pip install camelot-py[cv]
```
